# Changelog

All notable changes to promptkit are documented here.

## [0.2.0] - 2026-02-20

### Added
- Conditional blocks in templates
- Template inheritance (extend/override)
- GitHub Actions CI pipeline

## [0.1.0] - 2026-02-18

### Added
- Prompt template engine with variable injection and validation
- Template includes and composition
- Registry of reusable named templates
- Prompt chaining for multi-step LLM workflows
- CLI: `promptkit render`, `promptkit chain`, `promptkit list`
